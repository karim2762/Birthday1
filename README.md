# 💖 For My Best Friend — Interactive Birthday Web Experience

A beautifully crafted **single-file HTML tribute** designed to celebrate friendship and birthdays with elegant animations, heartfelt messages, and immersive storytelling.

> ✨ Built using pure HTML, CSS, and JavaScript — no frameworks required.

---

## 🌟 Features

- 🎉 **9 Interactive Pages** with smooth transitions
- 💫 **Animated Starfield Intro** with typing effect
- 📖 **Friendship Timeline** showcasing memorable moments
- 🖼️ **Interactive Photo Gallery** with flip cards and lightbox
- ❤️ **Reasons You're Amazing** flip-card section
- 📚 **Memory Book Slider** with animated memories
- 🎊 **Celebration Page** featuring confetti and balloons
- 🔒 **Secret Message Section** with unlock animation and audio support
- 🌸 **Friendship Promise** with elegant text animations
- 💝 **Beautiful Finale** with floating hearts and replay option
- 📱 Fully responsive design for desktop and mobile devices
- 🎨 Modern glassmorphism UI with premium gradients and animations

## 🛠️ Customization

You can personalize the experience by editing the `CONFIG` section inside the HTML file.

Update items such as:

- 👤 Friend's name
- 🖼️ Photo URLs or local images
- 🎵 Secret audio message
- 💌 Memories
- ❤️ Reasons
- ✨ Timeline events

Example:

```javascript
friendName: "Your Friend",

photos: [
  {
    src: "images/photo1.jpg",
    label: "Our favorite memory ❤️"
  }
],

secretAudio: "audio/song.mp3"
```
